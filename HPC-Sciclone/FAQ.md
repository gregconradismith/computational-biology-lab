# FAQ

## Is there an application that I can run locally that will allow me to drag and drop files between my computer and bora?

I do this using the command line, but I thought that there might be an application that does this.

Yes, there are two options:

1. Use Globus Personal Connect, which will add your local computer to the Globus site and let you drag and drop files to/from your computer and the cluster: https://www.wm.edu/offices/it/services/researchcomputing/using/filesandfilesystems/xfers/globus/personal/
2. Use a FUSE mount to mount a bora folder on your local computer, although this won't be drag and drop.
